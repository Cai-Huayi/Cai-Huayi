## 代表项目 · Selected work

- **[Smart Concrete](https://github.com/Cai-Huayi/auto-IOT-new-) / 智能混凝土振捣** — 连接传感、策略生成、GPIO/设备控制与三维点云可视化的模块化原型。A modular prototype connecting sensing, strategy generation, GPIO/device control, and 3D point visualization.
- **[Renewable Forecast](https://github.com/Cai-Huayi/advance_aviation_model) / 新能源功率预测** — 面向风电与光伏场站的 15 分钟预测比赛方案，包含三类 NWP/NetCDF 数据、物理特征工程、LightGBM 交叉验证和场站独立建模。A competition pipeline for 15-minute wind/solar forecasting with three NWP/NetCDF sources, physical feature engineering, LightGBM cross-validation, and station-specific models.
- **分时电价泵组调度 · Time-of-use Pump Scheduling** — 在 25,000 m³ 供水目标、泵组组合、分时电价和班次切换约束之间寻优，并进行高精度结果复算。An operations-research solution balancing delivery volume, pump combinations, tariffs, and shift-level switching constraints.
- **[Magic 4K](https://github.com/Cai-Huayi/magic_4K)** — 用 Python 解释四张牌数学魔术及其确定性终态。A Python exploration of the four-card mathematical magic trick and its deterministic final state.
- **TalkEase（开发中 · in development）** — 基于 Next.js 15、React 19 和 TypeScript 的 AI 沟通训练产品，覆盖学术、职场和关系场景。An AI communication coach for academic, workplace, and relationship scenarios.
- **Calculus Copilot（开发中 · in development）** — 融合 OCR、AI 反馈、OAuth 和课堂工作流的微积分学习浏览器扩展。A browser-based calculus learning assistant combining OCR, AI feedback, OAuth, and classroom workflows.
